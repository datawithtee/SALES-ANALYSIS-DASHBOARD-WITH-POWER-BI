# SALES-ANALYSIS-DASHBOARD-WITH-POWER-BI
Analyzed sales data to uncover revenue trends, top selling products, and key business insights. Using Power BI, data was cleaned and transformed in Power Query, DAX was used to create KPIs, and an interactive dashboard was built to support data driven decision making.

## Project Overview

This project focuses on analyzing sales data to uncover key business insights that support data-driven decision making. The analysis highlights revenue performance, profit trends, top selling products, and other important sales metrics through an interactive Power BI dashboard.

## Objectives

The main objectives of this project were to:

Analyze overall sales performance
Identify top performing products
Track revenue and profit trends over time
Build key performance indicators (KPIs) for business monitoring
Create an interactive dashboard for easy visualization and reporting

## Tools Used
Power BI – Data visualization and dashboard creation
Power Query – Data cleaning and transformation
DAX (Data Analysis Expressions) – Calculated measures,Calculated columns, and KPIs
Excel / CSV Dataset – Source data for analysis
Data Cleaning & Transformation

## Transformation 
The raw sales dataset was cleaned and prepared using Power Query by:

Removing duplicates and unnecessary columns
Checking for missing or null values
Standardizing data structure and formatting
Preparing the dataset for analysis and reporting
Key Metrics (KPIs)

## KPIs and Metrics
The dashboard tracks important business metrics including:

Total Revenue
Total Quantity of goods sold
Profit Margin
Sales Growth Trend
Top Selling Products
Product Category Performance
Top Selling City
Dashboard Features

## Visualization
The dashboard provides interactive visual insights through:

Revenue trend analysis
Product performance comparison
Profitability analysis
Sales breakdown by products
KPI summary cards
Interactive filters and slicers

Some insights generated from the analysis include:

Identification of best selling products driving revenue
Revenue trends across different periods
Product categories contributing the highest profit
Business areas requiring performance improvement
Project Outcome

This project demonstrates how sales data can be transformed into actionable insights using Power BI, enabling businesses to make better strategic and operational decisions.
